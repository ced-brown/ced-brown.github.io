PLAN

This is where the team and I planned out the project accordingly based off of the requirements document that was created. We took the time to brainstorm, research and prep. When doing this we made sure to pay attention to the goals and details that were layed out in the requirments document. To ensure our services met the requirements by the client we familiarized ourselfs with the project by analyzing it further and utilizing the AGILE principles. We kept an open line of communication with the client throughout the entire project which assisted us greatly in being able to satisfy the client expectations. If there were any changes or modifications on the client end we made sure to implement those changes accordingly.

DESIGN

SKETCH, WIREFRAME, PROTOTYPE, DEVELOP.
Here we established the architecture on pen and paper. After more analysis and research the wireframe and prototyping was designed with Adobe XD. Then came the development.

DEVELOP

DJANGO was the python based tool we used for development and imported Excel data sheets into the SQLite3 database. With AGILE principles in play we were able to catch errors and have better accuracy for things such as the project completion date. Using github as a source control allowed the client and teammates to report issues that could be fixed or implemented.
NGROK was used for testing the app publicly so that the client could test the app from any device that is connected to the internet.

TECH/PRINCIPLES

Adobe XD

Helped with the design of the software before development. We were able to implement details to wireframe and protype with this tool.

DJANGO

A framework that utilizes Python. We used the manage.py file that was created through DJANGO to run the website on our local host.

SQLite3

A lite version of SQL database. This implementation for a database allowed overall more effectiveness. We imported various excel spreadsheets as data sets for the information within the database.

GitHub

Our platform for source control. Great for collaborative projects and documentation.

Visual Studio

A software we used for editing most of our files. The files consist of python, html, excel, SQLite3. We also utilized Expression Web 4 as a tool for editing html files.

Ngrok

Used for deployment and testing. With ngrok as a temporary domain we were able to allow the clients to test the application from their own devices.
